Petunjuk Singkat: 
- website ini terdiri dari 2 halaman yaitu home dan Form
- menu Form akan mengarahkan pengguna untuk mengisi form
- penulisan alamat harus seusai dengan data di api (case sensitive) dan
  alamat ditulis berdasarkan nama kota dan kabupaten
- pengisian form berat menggunakan parameter gram menyesuaikan
  dengan fitur rajaongkir
- selesai melengkapi form, website akan menampilkan hasil yang ingin didapatkan

tambahan:
- jika data tidak terpanggil dan tabel tidak menampilkan nilai, ada dua kemungkinan kondisi yang terjadi:
  1. key api yang tertera telah melewati batas harian.
     jika kondisi ini terjadi, bisa hubungi saya dan akan saya berikan key API baru
  2. penulisan form alamat tidak sesuai dengan data di API
